# AItmosphere 🎵🎨

気象情報とテキストプロンプトを基に、AI (MusicGen, Stable Diffusion) が音楽と画像を生成し、動画を作成するWebアプリです。

-----

## 実行方法

### 依存ライブラリのインストール

```bash
pip3 install -r requirements.txt
```

### サーバーの起動

```bash
python3 app.py
```
初回起動時のみ、AIモデルのダウンロードが自動的に行われる。

### アプリケーションへのアクセス

サーバー起動後、ブラウザで下記のアドレスにアクセスする

`http://127.0.0.1:8000`
